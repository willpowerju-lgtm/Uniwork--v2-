## 投资论点 (Key Thesis)

### Bull Case (Our View)
1. AI推理/KVCache/RAG驱动数据中心NAND需求结构性爆发：CY2026数据中心exabyte增长预测从mid-20%连续上调至mid-70%+，数据中心首次超越移动成为NAND最大市场 ← EC FY26Q3 CEO + Bernstein 2026-05-28 conf

3. BiCS8 CBA节点领先+Stargate QLC平台构建企业SSD差异化壁垒：BiCS8 2Tb QLC裸片为行业最大，128TB/256TB/512TB路线图清晰，FQ3'26 TLC企业SSD占比25%，Stargate QLC Q4开始创收 ← Analyst Day 2025-02-11 + EC FY26Q3
4. 供给纪律+零债务+强FCF：行业无新增晶圆产能（新产能需数年），FQ3'26现金$37.35B零债务，FCF利润率49.7%，$6B回购授权，CapEx%下降趋势明确 ← EC FY26Q3 CFO + Citi 2026-05-19

### Bear Case / 空头逻辑
1. NAND周期性风险未完全消除：尽管NBM减少波动，历史上NAND供需反转往往比预期更快；当前ASP环比+140%为史上罕见，定价可能已近峰值 ← Bernstein 2026-05-01 + Nomura 2026-05-01
2. NBM合约可能约束比特价格上行空间：长期固定定价成分限制了现货市场极端上行的参与，市场盘后下跌~5-6%反映对NBM约束利润率增长的担忧 ← Nomura 2026-05-01 + Citi 2026-04-30
3. 公司信息披露质量有待提升：Bernstein指出公司信息披露较为混乱（messy disclosures），可能令注重质量的投资者望而却步 ← Bernstein 2026-05-01
4. 竞争对手产能扩张风险：Micron宣布新加坡新建NAND工厂（2H CY28初始产出），长期供给可能增加；YMTC等中国竞争对手持续在消费级市场扩展 ← Citi 2026-03-19 + EC FY25Q4 Q&A
5. 地缘政治+宏观风险：中美制裁可能冲击市场准入，AI资本开支放缓则直接影响数据中心NAND需求；能源成本上升压缩成本端 ← MS 2026-06-03 + Citi 2026-04-13

## 业务结构备忘 (Other Key Facts)

Sandisk Corporation（NASDAQ: SNDK）是全球领先的NAND闪存存储解决方案公司，2025年2月从Western Digital（WDC）分拆独立上市。公司通过与Kioxia的25年合资企业（JV延至2034年）运营位于日本的NAND FAB，产出全球约1/3的NAND比特。

**终端市场结构（FQ3'26，已重新分类）**：

| 终端市场 | FQ3'26 收入 | 占比 | 环比 | 关键产品 |
| ---- | ---- | ---- | ---- | ---- |
| Edge（PC/移动/嵌入式） | $36.63B | 62% | +118% | 客户端SSD, 嵌入式存储 |
| Datacenter | $14.67B | 25% | +233% | 企业SSD (TLC Gen5 + Stargate QLC) |
| Consumer | $8.20B | 13% | -10% | MicroSD, USB, 便携SSD |

**关键事实**：
- BiCS8 CBA（CMOS Bonded to Array）为当前主力节点，FQ3'26占比>50%
- 企业SSD占比特出货25%（TLC为主），Stargate QLC FQ4'26开始创收
- 5份NBM多年期供给协议，RPO $42B，金融担保$11B
